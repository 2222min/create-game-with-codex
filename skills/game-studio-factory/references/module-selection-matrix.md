# Module Selection Matrix

Use this matrix before editing any game code.

## By Business Goal

- Need direct revenue:
  - `payment-core`
  - `economy-wallet`
- Need retention and optimization:
  - `analytics-event-pipeline`
  - `liveops-season-pass`
- Need social virality:
  - `social-guild-and-leaderboard`

## By Game Type

- Idle RPG:
  - payment-core
  - economy-wallet
  - analytics-event-pipeline
  - social-guild-and-leaderboard
  - liveops-season-pass
- Pure skill arcade:
  - analytics-event-pipeline
  - social-guild-and-leaderboard
  - payment-core (cosmetics only)

## Minimum Revenue-Ready Set

1. `payment-core`
2. `economy-wallet`
3. `analytics-event-pipeline`
