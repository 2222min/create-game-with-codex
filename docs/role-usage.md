# Role Usage

Use these Korean prompts directly:

- `개발자1 모드로 이번 기능 아키텍처 설계해줘.`
- `개발자2 모드로 방금 아키텍처 기준으로 구현해줘.`
- `개발자3 모드로 지금 코드 강하게 리뷰해줘.`
- `디자이너1 모드로 필요한 무료 리소스 목록 뽑아줘.`
- `기획자1 모드로 다음 스프린트 계획 짜줘.`
- `기획자2 모드로 이 아이디어 상품성 검증해줘.`

Recommended sequence:

1. 기획자1
2. 기획자2
3. 개발자1
4. 개발자2
5. 개발자3
6. 디자이너1
