## Output Spec

- Format: PNG (transparent)
- Hero: 512x512
- Boss: 768x768
- Weapon icons: 128x128
- UI elements: 256x256 or 512x512 (9-slice ready)

## File Names

- hero: `hero_knight_<action>_<frame>.png`
- boss: `boss_<name>_<action>_<frame>.png`
- weapon: `weapon_sword_<rarity>_<index>.png`
- ui: `ui_<group>_<state>_<size>.png`
