# Market Skeptic v1

## Viability Score

- 7/10

## Strongest Risk

- 키우기 RPG 시장은 과포화라서 "그냥 성장한다"만으로는 선택받기 어렵다.

## Why It Can Fail

- 차별화 없는 성장 루프는 광고 유입 이후 빠르게 이탈
- 소셜 기능이 피상적이면 "자랑" 동기가 약함
- 과금이 전투 효율에 직접 연결되면 친구 그룹 내 불공정 인식으로 이탈

## Evidence Needed Before Full Build

1. 친구끼리 랭킹/배지 공유를 실제로 하는지
2. 3일 내 재접속(D1~D3) 유지되는지
3. 코스메틱 기반 과금 의향이 있는지

## Cheapest Validation Plan

1. Fake-door 테스트:
  - 배지 카드 공유 버튼과 시즌 패스 화면을 먼저 노출
  - 클릭률만 측정
2. 1주 소규모 테스트:
  - 10~30명 친구 그룹
  - 공유 횟수, 재접속, 플레이 시간 관찰
3. 결제 의향 설문:
  - "이 스킨 3,000원 구매 의향" 단일 문항 측정

## Go / Pivot / Stop Rule

- Go:
  - D1 >= 30%, 공유 버튼 CTR >= 20%
- Pivot:
  - D1 >= 20% but 공유 CTR < 10%
  - 소셜 루프/보상 구조 재설계
- Stop:
  - D1 < 15% and 재접속 동기 약함

## Recommendation

- MVP에서 전투보다 `자랑 가능한 성과 카드`를 핵심 기능으로 우선 개발
- 과금은 코스메틱과 편의 중심으로 제한해 친구 경쟁의 공정성 유지
