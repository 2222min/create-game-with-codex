# Milestone Cycle 01 (Builder Plan)

## Milestone Goal

- 친구 10명이 바로 테스트 가능한 "핵심 키우기 루프" 웹 빌드 완성

## Deliverables

1. 오프라인 보상 수령 UI
2. 자동 전투 + 보스 도전 1개
3. 성장 포인트 분배 (공격/체력/치명)
4. 결과 요약 카드(스코어, 클리어 층, 배지 후보)
5. 로컬 저장 기반 진행 유지

## Acceptance Criteria

- 첫 접속 후 3분 내 성장 체감 가능
- 보스 처치 실패/성공 피드백 명확
- 재접속 시 오프라인 보상 정상 반영
- 결과 카드 이미지를 스크린샷으로 바로 공유 가능

## Role Ownership

- 기획자1: 범위/우선순위 고정
- 기획자2: 상품성 체크와 검증 지표 관리
- 개발자1: 아키텍처와 테스트 전략
- 개발자2: 기능 구현과 결함 최소화
- 개발자3: 반박/리스크 리뷰
- 디자이너1: 무료 리소스 수급 및 스타일 가이드

## Immediate Next Task

- `web/` 디렉토리 생성 후 최소 프로토타입 부트스트랩
