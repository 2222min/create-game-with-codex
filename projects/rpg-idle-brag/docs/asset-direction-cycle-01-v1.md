# Asset Direction Cycle 01 v1

## Art Direction

- Theme: 네온 판타지 대시보드
- Tone: 밝고 경쟁적인 분위기, 정보 가독성 최우선
- Render Style: 2D 패널 + 그래디언트 배경 + 텍스트 중심 HUD

## Asset List (Current)

- 현재는 코드 기반 벡터/캔버스 도형으로 MVP 진행
- 외부 무료 리소스 없이 테스트 가능

## Integration Spec

- Canvas: `1024x576`
- HUD panel border contrast: WCAG 기준 대비 확보
- Typography: 타이틀 24~44px, 본문 12~16px
- 브래그 카드 영역은 항상 우측 하단 고정

## External Free Resource Request (Optional for Cycle 02)

- Needed Asset: 캐릭터 아이콘 세트(기본/희귀 스킨)
- Recommended Source: Kenney 또는 OpenGameArt 무료 팩
- License Type: CC0 또는 상업 이용 허용
- Target Folder: `assets/free-resources/inbox`
- File Naming Rule: `rpg_idle_icon_<theme>_<index>.png`
