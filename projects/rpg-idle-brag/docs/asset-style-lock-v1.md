# Asset Style Lock v1

## Locked Decisions

- Art Style: **Semi-Real**
- UI Tone: **Gold-Dark** (현재 프로토타입 테마 유지)

## Why this choice

- Semi-real 스타일은 보스 패턴 인지성과 장비 희귀도 차이를 표현하기 좋다.
- Gold-dark UI는 현재 화면 구조와 가장 자연스럽게 이어지고, 전환 비용이 낮다.
- 즉시 제작/통합 속도를 우선해 스타일 리스크를 줄인다.

## Visual Rules

- 캐릭터와 보스는 과도한 디테일보다 실루엣 가독성 우선.
- 금속, 가죽, 에너지 이펙트는 세미리얼 질감으로 표현.
- UI 프레임은 금속 테두리 + 어두운 내부 패널 유지.
- 텍스트 대비를 위해 배경 채도는 과하지 않게 제한.

## Immediate Art Targets

1. Hero set (semi-real): idle/attack/guard/dodge/hit
2. Boss set x3 (semi-real): idle/attack/hit
3. Sword icons x5 rarity (metal/material differentiation)
4. UI skin set (gold-dark panel, button, bar)
